Encrypted Wallet Load Test

This page will help you test loading the encrypted wallet and see console output.

  1. Open index.html in your browser
  2. Open Developer Console (F12)
  3. Click "Restore Wallet" button
  4. Select the encrypted wallet file: debug-service/examples/FIXED_ENC_alpha1qllh_axta7d_wif.txt
  5. Enter password: 123456
  6. Click "Load" button

Expected Behavior:

  • ✅ The restore modal should close immediately after loading
